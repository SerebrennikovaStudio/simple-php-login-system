# Simple PHP Login System

A lightweight and secure PHP login and registration app built with MySQL and HTML/CSS.

## Features
✅ User registration with password hashing  
✅ Login with session management  
✅ Logout function  
✅ Clean responsive design  

## Technologies
- PHP
- MySQL
- HTML
- CSS
